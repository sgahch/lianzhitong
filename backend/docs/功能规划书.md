# 廉智通管理后台 - 功能规划书

> 使用UltraThinking分析后的完整功能规划

---

## 1. 系统定位与目标

**系统定位**：廉政监督管理系统，面向纪检监察部门，提供案件全流程管理、廉政教育、监督检查等功能。

**核心目标**：
- 实现信访举报、案件立案、审查调查、案件审理、结案报告全流程数字化管理
- 支持日常业务（协助党委、纪律教育、监督检查、线索处置）的规范化管理
- 提供系统管理功能，确保数据安全和权限控制

### 2.2 信访举报 (Report)

**页面定位**：群众举报信息的登记、受理、分类、办理

#### 功能模块：
| 功能 | 说明 |
|------|------|
| 举报登记 | 新增举报信息（举报人、举报内容、举报方式） |
| 分类管理 | 按来源（信函、网络、电话）、性质分类 |
| 受理审核 | 审核举报信息，决定是否受理 |
| 转办分流 | 将举报件转办到相关部门 |
| 办理跟踪 | 跟踪举报件的办理进度 |
| 统计分析 | 举报数量、类型、趋势分析 |

#### 表格列设计：
| 列名 | 说明 |
|------|------|
| 举报编号 | 自动生成，如BJ20250127001 |
| 举报人 | 实名或匿名 |
| 举报方式 | 信函/网络/电话/来访 |
| 举报时间 | 格式：YYYY-MM-DD HH:mm |
| 举报类型 | 信访/投诉/举报 |
| 主要内容 | 摘要（超长省略） |
| 受理状态 | 待受理/已受理/已办结 |
| 办理进度 | 进度百分比 |
| 操作 | 查看/办理/转办 |

#### 布局设计：
```
┌─────────────────────────────────────────────────────┐
│  信访举报                          [+ 新增举报]     │
├─────────────────────────────────────────────────────┤
│  搜索举报人...    [全部状态 v]  [批量受理] [导出]   │
├─────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────┐   │
│  │  编号      │ 举报人 │ 方式 │ 时间    │ 状态  │   │
│  │  ───────  │ ────── │ ──── │ ─────── │ ───── │   │
│  │ BJ001     │ 张三   │ 网络 │ 01-27   │ 待受理│   │

├─────────────────────────────────────────────────────┤
```

### 2.3 案件立案 (Case Filing)

**页面定位**：违纪违法案件的立案登记与审批

#### 功能模块：
| 功能 | 说明 |
|------|------|
| 立案申请 | 新增立案申请（案件基本情况、被调查人信息） |
| 审批流程 | 分管领导审批、集体研究决定 |
| 案件编号 | 自动生成标准案件编号 |
| 证据管理 | 关联收集的证据材料 |
| 办案人员 | 指定办案人员 |
| 案件状态 | 初核中/已立案/已结案/已撤案 |

#### 表格列设计：
| 列名 | 说明 |
|------|------|
| 案件编号 | 标准格式，如LZ2025010001 |
| 案件名称 | 如"某局长涉嫌违纪案" |
| 被调查人 | 姓名+职务 |
| 案件类型 | 违纪/违法/职务犯罪 |
| 涉案金额 | 金额（元） |
| 立案时间 | 格式：YYYY-MM-DD |
| 办案人员 | 主办人/协办人 |
| 案件状态 | 初核/立案/结案/撤案 |
| 操作 | 查看/编辑/结案 |

## 3. 通用组件规范

### 3.1 列表页通用布局

```
┌────────────────────────────────────────────────────────────┐
│  页面标题                              [+ 主操作按钮]       │
├────────────────────────────────────────────────────────────┤
│  [搜索框]  [筛选条件]  [批量操作]  [导出]                  │
├────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────┐  │
│  │  表格内容                                          │  │
│  │  ...                                               │  │
│  └──────────────────────────────────────────────────────┘  │
├────────────────────────────────────────────────────────────┤
│        共 N 条  [< 1 2 3 >]  [每页 N 条 v]                 │
└────────────────────────────────────────────────────────────┘
```

### 3.2 详情页通用布局

```
┌────────────────────────────────────────────────────────────┐
│  返回  页面标题                                 [编辑]      │
├────────────────────────────────────────────────────────────┤
│  ┌── 基本信息 ─────────────────────────────────────────┐   │
│  │  字段1: 值1    字段2: 值2    字段3: 值3            │   │
│  └────────────────────────────────────────────────────────┘   │
│  ┌── 相关信息 ─────────────────────────────────────────┐   │
│  │  子表格/关联信息                                   │   │
│  └────────────────────────────────────────────────────────┘   │
│  ┌── 操作记录 ─────────────────────────────────────────┐   │
│  │  时间    操作人    操作内容                         │   │
│  └────────────────────────────────────────────────────────┘   │
└────────────────────────────────────────────────────────────┘
```

### 3.3 表单通用规范

- 必填字段标注红色星号
- 表单分组清晰，有标题分隔
- 支持表单校验
- 支持保存草稿
- 支持流程审批
