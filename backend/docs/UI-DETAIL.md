# 廉智通管理后台 - 界面详细描述

> 本文档可直接作为开发文档或 UI 设计参考

## 1. 全局布局架构 (Overall Architecture)

| 区域 | 说明 |
|------|------|
| 侧边栏 (Sidebar) | 固定于左侧，深色主题，用于系统级的模块导航 |
| 顶部栏 (Header) | 位于右侧上方，承载全局搜索、通知提醒及用户信息 |
| 主体内容区 (Main Content) | 位于右侧下方，采用浅灰色底色，以白色卡片容器承载具体业务数据 |

## 2. 左侧导航栏 (Sidebar Navigation)

### 2.1 顶部 Logo 区
- 系统图标 + 蓝色字体系统名称"廉智通"

### 2.2 一级菜单列表
垂直排列，每个项由"线性图标 + 标签文字"组成：
- 首页
- 信访举报
- 案件立案
- 调查取证
- 案件审理
- 结案报告

### 2.3 业务分组 (Group Labels)

**日常业务：**
- 协助党委（当前激活项，背景为深蓝色圆角矩形，文字图标反白）
- 纪律教育
- 监督检查
- 线索处置

**系统管理：**
- 权限管理
- 流程监控
- 法规库
- 系统集成

## 3. 顶部工具栏 (Global Header)

### 3.1 全局搜索
- 居中偏左位置
- 圆角长条搜索框
- 放大镜图标 + 占位文字

### 3.2 右侧操作区
- **消息中心**：铃铛图标，右上角带红色数字角标
- **个人中心**：圆形用户头像，旁有用户姓名"张科长"及下拉箭头

## 4. 主体内容区详细拆解 (Main Area Detail)

### 4.1 页面标题与全局操作
- **路径导航 (Breadcrumb)**：首页 > 日常业务 > 协助党委
- **主标题**：加粗文字"协助党委工作"
- **主操作按钮**：右上角蓝色加粗按钮"+ 新建任务"

### 4.2 数据过滤与工具条 (Filter & Tool Bar)
- **局部搜索**：左侧输入框，搜索任务名或负责人
- **筛选与功能键**：
  - 状态下拉筛选
  - 批量导出按钮（带导出图标）
  - 批量删除按钮（红色）

### 4.3 核心数据表格 (Data Table)
采用无边框平铺设计，列结构：

| 列名 | 说明 |
|------|------|
| 复选框列 | 用于批量勾选操作 |
| 任务名称 | 加粗文本，左对齐 |
| 负责人 | 标准文本 |
| 任务状态 | **状态标签（Badge）**展示 |
| 截止日期 | YYYY-MM-DD 格式 |
| 进度展示 | 水平进度条 + 百分比文字 |
| 操作列 | 查看、编辑、删除（蓝色链接） |

### 状态 Badge 颜色映射
| 状态 | 颜色 |
|------|------|
| 进行中 | 蓝色点 |
| 已完成 | 绿色点 |
| 已逾期 | 红色点 |

### 4.4 分页控件 (Pagination)
- 位于表格右下方
- 左侧显示"每页显示 [20] 条"的切换下拉框
- 右侧为数字翻页器及前进/后退箭头

## 开发建议

推荐使用 **Ant Design Vue** 库，可几乎 1:1 还原此效果：

| 功能 | 组件 |
|------|------|
| 整体布局 | `a-layout` |
| 侧边栏 | `a-menu` |
| 核心表格 | `a-table` |
| 进度条 | `a-progress` |
| 面包屑 | `a-breadcrumb` |
| 搜索框 | `a-input-search` |
| 下拉选择 | `a-select` |
| 按钮 | `a-button` |
| 弹窗 | `a-modal` |
| 消息提示 | `message` / `Modal` |
