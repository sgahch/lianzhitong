# 廉智通系统前端使用手册

## 一、系统概述

廉智通是一个纪检监察业务管理系统，前端采用 Vue 3 + Ant Design Vue 4.x 构建，提供完整的日常业务办理、监督检查、纪律教育等功能。

### 技术栈
- **框架**: Vue 3 (Composition API + TypeScript)
- **UI 组件库**: Ant Design Vue 4.x
- **状态管理**: Pinia (按需引入)
- **路由**: Vue Router 4
- **网络请求**: Axios + JWT 认证
- **构建工具**: Vite

### 3. 日常业务模块

#### 3.1 信访举报 (ReportPage)
**路由**: `/report`

**业务流程**: 受理群众来信来访 → 登记举报信息 → 分类处理 → 转办/直查

| 功能 | 操作 |
|------|------|
| 新增举报 | 点击"新增举报"按钮，填写举报信息 |
| 查看详情 | 点击"查看"查看举报详情 |
| 受理举报 | 待受理状态的举报可点击"受理" |
| 转办举报 | 已受理的举报可转办到其他单位 |
| 批量受理 | 勾选多条记录批量受理 |
| 导出 | 导出举报列表 |

**状态流转**: `待受理` → `已受理` → `已转办/已办结`

---

#### 3.2 案件立案 (CaseFilingPage)
**路由**: `/case-filing`

**业务流程**: 线索初核 → 立案审批 → 审查调查 → 审理 → 结案

| 功能 | 操作 |
|------|------|
| 新建立案 | 点击"新建立案"，选择线索或直接创建 |
| 查看案件 | 点击"查看"查看案件详情 |
| 编辑案件 | 点击"编辑"修改案件信息 |
| 结案操作 | 已立案状态的案件可点击"结案"跳转到结案页 |

**案件类型**: 违纪 / 违法 / 职务犯罪

**案件状态**: `初核中` → `已立案` → `已结案` → `已撤案`

---

#### 3.3 审查调查 (InvestigationPage)
**路由**: `/investigation`

**业务流程**: 收集证据 → 固定证据 → 证据保管

| 功能 | 操作 |
|------|------|
| 新增证据 | 点击"新增证据"，填写证据信息并上传附件 |
| 查看证据 | 点击"查看"查看证据详情 |
| 下载附件 | 点击"下载"下载证据材料 |

**证据类型**:
| 类型 | 说明 |
|------|------|
| document | 书证 |
| material | 物证 |
| testimony | 证人证言 |
| audio | 视听资料 |
| expert | 鉴定意见 |

**关联案件**: 证据需要关联到具体的案件

---

#### 3.4 案件审理 (TrialPage)
**路由**: `/trial`

**业务流程**: 受理案件 → 审理谈话 → 形成审理意见 → 审结

| 功能 | 操作 |
|------|------|
| 受理案件 | 点击"受理案件"，选择要审理的案件 |
| 开始审理 | 待审理状态的记录可点击"开始审理" |
| 审结操作 | 审理中的记录可点击"审结"完成审理 |
| 查看详情 | 点击"查看"查看审理详情 |

**状态流转**: `待审理` → `审理中` → `已审结`

---

## 四、通用操作说明

### 表格操作
- **搜索**: 输入关键词实时筛选
- **筛选**: 通过下拉选择框按类型、状态等筛选
- **分页**: 支持切换每页条数，跳转到指定页
- **导出**: 点击"导出"按钮导出当前筛选的数据

### 模态框操作
- **新增/编辑**: 填写表单后点击"确定"保存
- **取消**: 点击"取消"或点击遮罩层关闭
- **表单验证**: 必填项未填写会提示错误

### 状态标识
| 状态 | 标识颜色 | 说明 |
|------|----------|------|
| 待处理 | 橙色 | 待受理/待审理/待开始 |
| 进行中 | 蓝色 | 处理中/进行中 |
| 已完成 | 绿色 | 已受理/已审结/已完成 |
| 已逾期/异常 | 红色 | 逾期/异常状态 |

## 六、常见问题

### Q1: 退出登录后为什么还能看到页面?
A: 请检查浏览器 localStorage 是否清除了 `lzt_access_token`，如有问题请联系管理员。

### Q2: 批量操作没有反应?
A: 请先勾选表格左侧的复选框，选择要操作的记录后再执行批量操作。

### Q3: 文件上传失败?
A: 请检查文件大小是否超过限制（默认 10MB），格式是否支持。

### Q4: 如何切换深色模式?
A: 在页面右上角用户下拉菜单中选择"系统设置"（预留功能）。

### API 认证

所有 API 请求通过 Axios 发送，自动携带 JWT Token:

```typescript
// 请求拦截器自动添加 Token
request.interceptors.request.use((config) => {
  const token = localStorage.getItem('lzt_access_token')
  if (token && config.headers) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

// 响应拦截器处理 Token 刷新
// 401 错误时自动尝试刷新 Token
```
